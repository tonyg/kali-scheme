Cells

These hold a single value and are useful when a simple indirection is required. The system uses these to hold the values of lexical variables that may be set!.

  • (cell? value) -> boolean 
  • (make-cell value) -> cell 
  • (cell-ref cell) -> value 
  • (cell-set! cell value)
